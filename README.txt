🐰 Algomon Website - CA1

Welcome to Algomon, a fictional website created as a client-side programming project (CA1) for the CCT Year 1 course.

This project was built using only **HTML, CSS, and Vanilla JavaScript** — no external frameworks or libraries - to meet the requirements.

🌐 Technologies Used

- **HTML5** – Semantic, multi-page layout
- **CSS3** – Responsive design, animations, Flexbox layout
- **JavaScript (Vanilla)** – DOM manipulation, events, and interactivity

---

📁 Project Pages

| Page              | Description                                      |
|-------------------|--------------------------------------------------|
| `index.html`      | Homepage with interactive button and form        |
| `about.html`      | Team info (coordinators and developers)          |
| `gallery.html`    | Bunny-themed image gallery                       |
| `characters.html` | Main Algomon characters displayed in card format |
| `news.html`       | Latest game news and updates                     |
| `battle.html`     | Bonus page explaining the battle system          |
---

💡 JavaScript Features

- ✅ `click` event on button → displays a fun alert
- ✅ `hover` effect on images → glow effect using JS
- ✅ `keydown` event → secret "A" key changes background
- ✅ Form submission → with input validation and feedback message
---

🖼️ Images

Character images were generated using AI (DALL·E) to represent fictional game creatures.
